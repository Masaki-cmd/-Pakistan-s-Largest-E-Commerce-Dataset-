# データ分析プロジェクト: 不動産価格要因と手書き数字認識データ探索

## プロジェクト概要

本プロジェクトは、提供された2種類のデータセット（カリフォルニア州の住宅価格データと手書き数字認識のためのMNISTデータ）に対して、厳格なデータ分析ワークフロー（Data Understanding, Data Cleaning, Adaptive EDA, Statistical Validation, Insight Generation, Recommendation, Impact x Effort）を適用し、データ駆動型の意思決定に繋がる構造化されたインサイトと具体的な施策を導出することを目的としています。実在データのみに基づき、推測を排除した分析を実施しました。

## データ概要

### 1. MNISTデータセット (mnist_train_small.csv, mnist_test.csv)

*   **内容**: 手書き数字の画像データとそれに対応するラベル（数字）。
*   **形式**: ヘッダーなしのCSVファイル。最初の列がラベル、残りの列がピクセル値（0-255）。
*   **ファイルサイズ**: `mnist_train_small.csv` (19999行, 785列), `mnist_test.csv` (10000行, 785列)
*   **データ型**: 全て`int64`。
*   **欠損値**: なし。
*   **重複行**: なし。
*   **主キー候補**: 単一列での主キー候補なし。全ての列の組み合わせが一意であるため、複合主キー候補。
*   **分析可能な粒度**: 手書き数字の画像データ（各ピクセルの値とラベル）の一行。

### 2. California Housingデータセット (california_housing_train.csv, california_housing_test.csv)

*   **内容**: カリフォルニア州の特定の地域（ブロック）における住宅に関する統計データ。
*   **形式**: ヘッダー付きのCSVファイル。
*   **ファイルサイズ**: `california_housing_train.csv` (17000行, 9列), `california_housing_test.csv` (3000行, 9列)
*   **データ型**: `float64` (数値データ)。
*   **欠損値**: なし。
*   **重複行**: なし。
*   **主キー候補**: 単一列での主キー候補なし。全ての列の組み合わせが一意であるため、複合主キー候補。
*   **分析可能な粒度**: カリフォルニア州の特定の地域（ブロック）における住宅に関する統計データの一行。

## 分析手法

1.  **Data Understanding (データ理解)**: 各データセットの行数・列数、データ型、欠損率、重複行数、主キー候補、分析可能な粒度を調査。
2.  **Data Cleaning (データクレンジング)**: 欠損値処理（今回は該当なし）、外れ値（IQR法）、業務ロジックに基づく異常値チェックを実施。California Housingデータでは多数の外れ値と異常値（例: `households` > `population`）を検出。
3.  **Adaptive EDA (探索的データ分析)**: データ構造に応じて分析項目を自動選択。分布分析、グループ比較分析、構造分析（相関分析、寄与度分析）を実施。時系列データではないためトレンド分析は適用外。
    *   **MNIST**: ラベル分布、ピクセル値統計、ラベルごとの平均ピクセル値、分散の大きいピクセルを分析。
    *   **California Housing**: `median_house_value`の分布、主要特徴量（`median_income`, `housing_median_age`）の分布、築年数区分別の中央住宅価格、目的変数との相関を分析。
4.  **Statistical Validation (統計的検証)**: EDAで導出された仮説の中から、ユーザー指定の仮説を統計的に検証。
    *   **仮説5**: `median_income`が`median_house_value`の主要な予測因子である。単回帰分析（OLS）で検証。
    *   **仮説6**: 築年数が若いほど、中央住宅価格は高い傾向がある。一元配置分散分析（ANOVA）で検証。
5.  **Insight Generation (インサイト生成)**: 統計的検証の結果に基づき、「事実（データ）」「構造（なぜそうなるか）」「意味（ビジネス影響）」「リスク」の形式で構造化されたインサイトを生成。
6.  **Recommendation (施策提言)**: インサイトに基づき、Growth Lever（増収）およびRisk Lever（構造リスク）に分類された具体的な施策を提案。各施策には、根拠EDA番号、期待効果、実行難易度、成功判定KPIを付与。
7.  **Impact × Effort Analysis (インパクト×エフォート分析)**: 提案された施策について、Impact (High/Medium/Low) と Effort (High/Medium/Low) を評価し、実行優先順位を決定。

## 主要インサイト

### インサイト1: 所得水準が住宅価格に強く影響する
*   **事実（データ）**: `median_income`は`median_house_value`の変動の約47.9%を説明し、統計的に非常に有意な予測因子です (p値 < 0.001)。
*   **構造（なぜそうなるか）**: 高所得層が多く住む地域では、住宅に対する購買力が高く、需要と供給のバランスが住宅価格を押し上げる傾向にあります。
*   **意味（ビジネス影響）**: 不動産開発や投資において、地域の中央所得は最も信頼性の高い指標の一つであり、高所得地域をターゲットとすることで高い収益性が期待できます。
*   **リスク**: 所得以外の要因を考慮しないと、予測の精度が低下する可能性があります。

### インサイト2: 築年数と住宅価格の関係は複雑であり、単純な線形傾向ではない
*   **事実（データ）**: 築年数区分によって`median_house_value`の平均値に統計的に有意な差がありますが (p値 < 0.001)、最も高い平均価格を示すのは「築年数50年以上」の区分でした。これは「築年数が若いほど高い」という単純な仮説とは異なります。
*   **構造（なぜそうなるか）**: 築浅物件のプレミアムに加え、築古物件の中には歴史的価値、再開発ポテンシャル、希少な立地条件などにより高い価値を持つものがあるためです。
*   **意味（ビジネス影響）**: 不動産市場は築年数だけで一律に評価すべきではなく、「築浅プレミアム」と「築古レガシー」といった異なるセグメントを認識した多様な戦略が必要です。
*   **リスク**: 築年数のみに注目した評価は、市場の現実を反映せず、誤った投資判断につながる可能性があります。

## 施策

### 1. 高所得地域をターゲットとした住宅開発・投資戦略の最適化
*   **根拠EDA番号**: California Housing EDA - 構造分析: 目的変数との相関（仮説5の統計的検証）
*   **期待効果**: `median_income`が高い地域に不動産開発や投資を集中させることで、高い販売価格や賃料収入が見込め、収益性を向上させることが期待されます。
*   **実行難易度**: Medium
*   **成功判定KPI**: ターゲット地域における新規開発物件の平均販売価格、投資案件の平均ROI、顧客層の平均`median_income`。

### 2. 築年数別セグメントに応じたマーケティング戦略の策定
*   **根拠EDA番号**: California Housing EDA - 比較分析: 築年数区分別の中央住宅価格（仮説6の統計的検証）
*   **期待効果**: 各築年数セグメントの特性と顧客ニーズに合致したマーケティングメッセージやチャネル戦略を展開することで、販売機会を最大化し、顧客エンゲージメントを向上させることが期待されます。
*   **実行難易度**: Medium
*   **成功判定KPI**: 各築年数セグメントにおける広告反応率、販売成約率、顧客満足度調査結果。

### 3. 築古高価格帯物件に対する詳細なデューデリジェンスプロセスの導入
*   **根拠EDA番号**: California Housing EDA - 比較分析: 築年数区分別の中央住宅価格（仮説6の統計的検証）
*   **期待効果**: 築年数50年以上の高価格帯物件に潜在するリスクを事前に特定し、予期せぬコストや投資損失を最小限に抑えることで、長期的な投資の安定性を確保します。
*   **実行難易度**: High
*   **成功判定KPI**: 築古物件取得後の予期せぬ大規模修繕費用の発生率、法的紛争発生件数、デューデリジェンスレポートにおけるリスク指摘事項の平均件数。

*   **取得方法**: 社内データ（過去プロジェクト実績）、市場調査、専門家へのヒアリング、A/Bテストやパイロットプロジェクト。
*   **代替指標**: ターゲット地域における住宅販売数の増加率、コンバージョン率の改善、重大リスク発生件数の減少率など。

## 限界

*   **データセットの制約**: 分析に使用したデータセットは、特定の時点、特定の地域（カリフォルニア州）に限定されています。この結果が全ての地域や時点に普遍的に適用されるとは限りません。
*   **相関と因果関係**: 統計的検証で相関関係が確認された場合でも、それが直接的な因果関係を示すものではありません。他の潜在的な交絡因子が存在する可能性があります。
*   **モデルの単純性**: 仮説検証には単回帰分析やANOVAを使用しており、データに存在する複雑な非線形関係や多変量関係を完全に捉えきれていない可能性があります。
*   **ビジネス理解の初期段階**: 現時点ではビジネス課題が具体的に指定されていないため、一般的なビジネス示唆に留まっています。具体的な課題が定義されれば、より深い分析が可能です。
*   **外れ値処理**: IQR法による外れ値検出は実施しましたが、これらの外れ値の具体的な処理（除去、変換など）は行っていません。モデル構築時にはこれらの影響を考慮する必要があります。
